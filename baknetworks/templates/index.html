Welcome to an online portfolio

Hi. My name is Benjamin Kavanagh, and I am an aspiring developer. Thank you for visiting my portfolio.

This website has been build using Python, HTML, CSS, and (the tiniest bit of) Javascript, and is hosted on Heroku using a single dyno. While its primary intention is to showcase the workings of interactive user interfaced Neural Networks, it also demonstrates full stack development, including:

Back-end:

  • Web Framework Development using Flask.
  • Database Management using SQLAlchemy (SQLite).
  • User Authentication through OAuth.
  • Web Application Security using HTTPS, SSL, TLS, and SCPs.
  • Web Server Gateway Interfacing using GUnicorn.
  • Neural Network Deployment using TensorFlow.
  • Form control and security using WTForms.

Front End

  • Web development through HTML and CSS (utilising Bootstrap) and Javascript.

This project is completely open source, and I encourage you to visit its repository. All code is maintained, commented, and organised. Each Neural Network has its own repository, for CLI training and testing, including Shakesbot, Tolstoybot, and COVIDXNet.

The Dev Log page provides entries which detail the progression and development process of this portfolio, discussing the challenges faced, and how particular decisions were made.

The Neural Network Page showcases some of the Neural Networks I've developed. The Recurrent Neural Networks are interactive, while the Convolutional Neural Network is a demonstration. (See Dev Log #2 for a more in depth discussion on this).

It is not a requirement that you log in. This functionality mere acts as a demonstration of user management, OAuth autherisation with a third party, database management, and web security. The website is secured using SSL, TLS, and HTTPS, with an extremely secure custom Security Content Protocol, and no passwords are saved within the server's database. As such, the only benefit to logging in is to save RNN output into a history database, and to leave a comment for other users to see. All primary functionality is available to everyone.

Thank you for taking your time to visit my portfolio, and I hope you appreciate the passion and joy this project has brought to me.
